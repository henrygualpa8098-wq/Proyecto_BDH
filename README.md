# Evaluación de las políticas públicas de empleo juvenil en Ecuador mediante una arquitectura multiagéntica

## Descripción del proyecto

Este proyecto tiene como objetivo evaluar los resultados de las políticas públicas relacionadas con el empleo juvenil en Ecuador mediante el uso de una arquitectura compuesta por agentes y subagentes de inteligencia artificial.

El proyecto analiza indicadores laborales juveniles, procesa información estadística y presenta resultados mediante un dashboard interactivo.

---

# Modalidad del proyecto

Evaluación de resultados o impacto de una política pública.

---

# Problema público

El desempleo juvenil representa un desafío social y económico en Ecuador debido a las dificultades que enfrentan los jóvenes para acceder a empleos adecuados, formales y estables.

El proyecto analiza la evolución de indicadores laborales juveniles para generar evidencia que permita formular recomendaciones de política pública.

---

# Objetivo general

Evaluar los resultados de las políticas públicas de empleo juvenil en Ecuador mediante análisis de datos y una arquitectura multiagéntica de inteligencia artificial.

---

# Arquitectura multiagéntica

El sistema está compuesto por:

## Agente Coordinador
Organiza el proyecto y coordina las actividades.

## Agente de Política Pública
Analiza instituciones, actores y ciclo de políticas públicas.

## Agente de Datos y Metodología
Gestiona datos, variables e indicadores.

## Agente de Programación y Visualización
Desarrolla análisis, gráficos y dashboard.

## Subagente Verificador
Comprueba fuentes y calidad de información.

## Subagente Crítico
Detecta errores metodológicos y técnicos.
