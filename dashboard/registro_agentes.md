# Registro de Agentes y Subagentes de Inteligencia Artificial

## Proyecto:
Evaluación de las políticas públicas de empleo juvenil en Ecuador mediante una arquitectura multiagéntica.

## 1. Agente Coordinador

### Función:
Coordinar las actividades generales del proyecto y organizar la interacción entre los agentes especializados.

### Tareas asignadas:
- Organizar el flujo de trabajo.
- Distribuir responsabilidades entre agentes.
- Revisar los resultados obtenidos.
- Coordinar la toma de decisiones.

### Prompt principal:
"Actúa como director de un proyecto de política pública. Coordina agentes especializados, revisa resultados y organiza las actividades necesarias para cumplir los objetivos del proyecto."

### Resultado obtenido:
Permitió estructurar la arquitectura multiagéntica y coordinar las actividades del análisis.

---

# 2. Agente Especialista en Políticas Públicas

### Función:
Analizar el problema público, instituciones, actores y ciclo de la política pública.

### Tareas asignadas:
- Identificar el problema del desempleo juvenil.
- Analizar instituciones responsables.
- Identificar actores involucrados.
- Revisar fases de la política pública.

### Prompt principal:
"Analiza un problema público relacionado con el desempleo juvenil en Ecuador, identificando instituciones, actores y elementos del ciclo de política pública."

### Resultado obtenido:
Definición del problema público y estructura institucional del proyecto.

---

# 3. Agente de Datos y Metodología

### Función:
Gestionar información estadística y definir la metodología de análisis.

### Tareas asignadas:
- Seleccionar variables.
- Organizar la base de datos.
- Definir indicadores.
- Proponer métodos de análisis.

### Prompt principal:
"Actúa como especialista en datos públicos. Selecciona indicadores adecuados para evaluar políticas de empleo juvenil."

### Resultado obtenido:
Creación de la base de datos con indicadores laborales juveniles.

---

# 4. Agente de Programación y Visualización

### Función:
Desarrollar código y herramientas visuales para presentar resultados.

### Tareas asignadas:
- Programar análisis en Python.
- Generar gráficos.
- Crear dashboard interactivo.

### Prompt principal:
"Desarrolla soluciones de programación para analizar datos de política pública y generar visualizaciones claras."

### Resultado obtenido:
Creación del análisis Python y dashboard mediante Streamlit.

---

# 5. Subagente Verificador

### Función:
Comprobar la calidad de las fuentes y datos utilizados.

### Tareas asignadas:
- Revisar información estadística.
- Verificar referencias.
- Detectar inconsistencias.

### Prompt principal:
"Verifica la calidad, origen y consistencia de los datos utilizados en un proyecto de política pública."

### Resultado obtenido:
Validación de fuentes e indicadores utilizados.

---

# 6. Subagente Crítico

### Función:
Detectar errores conceptuales, metodológicos y técnicos.

### Tareas asignadas:
- Revisar resultados.
- Identificar limitaciones.
- Proponer mejoras.

### Prompt principal:
"Actúa como evaluador crítico de un proyecto de política pública e identifica posibles errores y mejoras."

### Resultado obtenido:
Revisión metodológica y recomendaciones de mejora.

---

# Validación humana

El estudiante revisó los resultados generados por cada agente, verificó la coherencia de la información y tomó las decisiones finales sobre metodología, datos y presentación del proyecto.

# Decisiones aceptadas

- Uso de indicadores laborales juveniles.
- Análisis descriptivo de datos.
- Implementación de dashboard interactivo.

# Decisiones rechazadas

- Utilizar información sin fuente verificable.
- Generar conclusiones sin evidencia estadística.
